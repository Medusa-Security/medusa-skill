# Medusa Skills

A curated collection of high-quality AI coding agent skills for software engineering, security, AI research, cloud infrastructure, and quantitative finance.

---

## Installation

Install any skill using:

```bash
npx skills@latest add Medusa-Security/medusa-skill --skill <skill-name>
```

### Example

```bash
npx skills@latest add Medusa-Security/medusa-skill --skill medusa-security
```

or

```bash
npx skills@latest add Medusa-Security/medusa-skill --skill hermes
```

Replace `<skill-name>` with any of the skills listed below.
