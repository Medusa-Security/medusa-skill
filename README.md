## What is Medusa Skills?

Medusa Skills is a curated collection of portable skill files that extend AI coding agents with specialized expertise.

Each skill is self-contained and can be installed independently using the `--skill` flag.

Supported agents include:

- Claude Code
- OpenAI Codex
- Cursor
- GitHub Copilot
- Antigravity
- Continue
- Windsurf
- Any agent supporting the Skills specification

Every skill is designed around a single domain and teaches the AI how senior engineers approach that problem.

## Featured Skill ‚ÄĒ Medusa Security

The flagship skill in this repository is **medusa-security**.

It performs exhaustive security reviews across:

- Secrets & Credentials
- Injection Vulnerabilities
- Authentication
- Authorization
- API Security
- AI / LLM Security
- Supply Chain
- Infrastructure as Code
- Runtime Security
- Business Logic
- Cryptography
- Operational Security

Every finding includes:

- Severity
- CWE
- Attack scenario
- Financial breach estimate
- Diff-ready remediation
